🧑🏻‍💻 소개

안녕하세요. 연성대학교 컴퓨터소프트웨어학과 1학년에 재학 중인 신지안입니다. 저는 21살이고, 평소에 웹 개발에 관심이 많습니다.

HTML, CSS 같은 기본 기술부터 시작해서 JavaScript로 활용성 높은 웹페이지를 만드는 걸 배우고 있어요!

🏫 학교 생활

  • 컴퓨터소프트웨어학과 1학년 B반
  • 현재 수강 중인 과목 : 인터넷 프로그래밍, 파이썬, C언어 등
  • 발표 수업과 참여를 통해 열심히 배우고 있습니다.

💻 기술 역량

기술 경험 및 설명
HTML5 웹 페이지 구조 설계와 시멘틱 태그 활용
CSS3 디자인 구현 레이아웃
JavaScript 간단한 기능 구현과 동적인 사용자 경험 구현
Git / GitHub 버전 관리 및 코드 공유 경험 있음
Figma UI 시안 제작 및 팀 프로젝트 협업

🌈 취미와 성격

평소에 IT 관련 콘텐츠나 유튜브 영상을 자주 보고, 재밌어보이는 기능이 있으면 직접 따라 만들어보는 걸 좋아합니다. 가볍게 깃허브에 기록도 남기고 있습니다.

성격은 꼼꼼한 편이고, 맡은 일은 끝까지 처리하려고 합니다. 팀 프로젝트를 할 땐 조율하고 의견 조합하는 걸 좋아하는 편입니다.

🚀 앞으로의 목표

단기 목표는 배운 내용을 토대로 포트폴리오 웹사이트를 만들어보는 게 목표입니다. 단순히 예쁨만 강조되는 페이지가 아니라, 사용자 입장에서 보기 편리한. 가독성 좋은 구조를 생각하며 만들고 싶습니다.

장기적으로는 프론트엔트 / 백엔드 개발자로 성장해서 다양한 프로젝트와 웹사이트를 개발해 보며, 사용자에게 유용한 서비스를 만들어보고 싶습니다.

  • 웹 개발 기초 익히기 (HTML, CSS, JS)
  • React 등 외부 프레임워크도 차근차근 배워보기
  • 작은 토이 프로젝트부터 시작하기
